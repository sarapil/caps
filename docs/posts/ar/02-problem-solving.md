## يعمل مع ERPNext الحالي لديك

CAPS لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/caps
bench --site your-site install-app caps
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
