# 🛡️ CAPS — Fine-Grained Access Control for Frappe Apps

> **Fine-Grained Access Control for Frappe Apps** — Open source, self-hosted, integrated with ERPNext.

---

## Why CAPS?

Businesses in the Security space face real challenges:
- ❌ Frappe's built-in roles are too coarse
- ❌ No field-level restrictions out of the box
- ❌ Permission changes not auditable
- ❌ Complex multi-app permission requirements

**CAPS solves all of this** — natively inside Frappe/ERPNext.

## ✨ Key Features

- ✅ Capability-Based Permissions
- ✅ Field-Level Access Control
- ✅ Role → Job → User Hierarchy
- ✅ Visual Permission Editor
- ✅ Permission Audit Trail
- ✅ Cross-App Permission Management
- ✅ API Permission Enforcement
- ✅ Dynamic Permission Rules
